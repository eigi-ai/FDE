# FastAPI Basics

This folder is reserved for the next stage of the FDE training module.

It will be used after the learner completes:

1. Python programming fundamentals
2. Object-Oriented Programming basics
3. API and backend fundamentals

Planned topics for this module include:

- what FastAPI is
- why FastAPI is used
- creating the first FastAPI app
- routes and endpoints
- query parameters and path parameters
- request body handling
- validation with Pydantic
- response models
- async route handlers
- basic project structure
