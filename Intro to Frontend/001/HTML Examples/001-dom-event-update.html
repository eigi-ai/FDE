DOM Event Update

Use this to explain: the browser creates a live DOM tree, JavaScript listens for events, and DOM changes repaint the page.

Live page

Original DOM text
1. Browser has HTML in the DOM.
2. User clicks the button.
3. JavaScript event handler runs.
4. JavaScript changes textContent.
5. Browser repaints the new text.

Code idea

const message = document.querySelector("#message");
const button = document.querySelector("#changeButton");

button.addEventListener("click", () => {
  message.textContent = "Updated DOM text";
});
