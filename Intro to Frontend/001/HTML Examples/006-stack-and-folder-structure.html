Stack And Folder Structure

Use this as a visual teaching aid for stack selection and project organization. Tools should match product constraints.

Choose stack from constraints

Simple learning app HTML, CSS, JavaScript. Best when the goal is understanding the flow.
Small interactive app React + Vite. Useful when the UI has reusable components and state.
Internal dashboard React + TypeScript + forms + API state tools. Good for tables, filters, auth, and workflows.
Public SEO site Next.js or Astro. Useful when crawlable content and fast first load matter.
Rule: pick the smallest stack that handles the real product constraints.
