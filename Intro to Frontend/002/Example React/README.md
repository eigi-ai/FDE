# FDE React 002 Concepts Demo

This is a Vite React TypeScript teaching app for the FDE Frontend 002 session.

It demonstrates components, props, `useState`, events, conditional rendering, lists, forms, `useEffect`, API-state handling, custom hooks, browser DevTools, storage, cookies, and Vite React TypeScript folder structure.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## DevTools Teaching

- Console: click `Log demo state`
- Network: click `Fetch success` and `Fetch failure`
- Application: click `Write storage and cookie`, then inspect Local Storage, Session Storage, and Cookies
