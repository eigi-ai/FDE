# Activity - Build Static Components

## Goal

Practice JSX and component creation.

## Task

Create a Vite React app and build a static contact screen.

Setup:

```bash
npm create vite@latest react-components-activity -- --template react
cd react-components-activity
npm install
npm run dev
```

## Required Components

```text
App
PageHeader
ContactCard
ContactList
EmptyState
```

## Requirements

- Use JSX.
- Use `className`.
- Use at least one `<label htmlFor="...">`.
- Use component names with capital letters.
- No state yet.
- No API yet.
