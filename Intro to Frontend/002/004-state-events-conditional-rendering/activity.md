# Activity - Counter, Toggle, And UI States

## Goal

Practice `useState`, events, and conditional rendering.

## Task 1: Counter

Build a counter with:

- Increase
- Decrease
- Reset

Must use:

```jsx
const [count, setCount] = useState(0);
```

## Task 2: Help Toggle

Add a button that shows/hides help text.

Must use:

```jsx
const [showHelp, setShowHelp] = useState(false);
```

## Task 3: Manual UI State Switcher

Create buttons for:

```text
Idle
Loading
Success
Error
Empty
```

Clicking each button should show a different UI message.

## Demo Questions

```text
Which state variable changes when you click Increase?
Which state variable controls the help message?
How does React know what UI to show?
```
