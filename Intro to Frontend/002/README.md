## Learning Goal

By the end, you should be able to build and explain a React app using:

- components
- JSX
- props
- `useState`
- events
- conditional rendering
- lists and keys
- controlled forms
- `useEffect`
- API calls
- custom hooks
- browser Developer Tools
- Console and Network debugging
- local storage, session storage, and cookies
- loading, error, empty, and success states
