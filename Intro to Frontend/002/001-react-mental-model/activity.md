# Activity - Identify React Components

## Goal

Practice thinking in components before writing code.

## Task

Pick any familiar app screen, such as:

- CRM contacts page
- weather search app
- quotes app
- food nutrition search
- task manager

Break the screen into components.

## Example

For a CRM contacts page:

```text
App
  ContactsPage
    PageHeader
    SearchBar
    ContactForm
    ContactList
      ContactCard
    EmptyState
    ErrorMessage
```

## Deliverable

Create a short `component-plan.md` with:

1. App name
2. Component tree
3. What each component displays
4. Which components may need state later

## Demo Question

```text
Why did you split the screen into these components?
```
