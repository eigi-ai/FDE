# Activity - Explain Git And GitHub With A Tree

## Goal

Practice explaining Git and GitHub without commands first.

## Task

Create a short file named:

```text
git-github-mental-model.md
```

## Requirements

Your file must explain:

- what Git is
- what GitHub is
- why Git is useful
- why GitHub is useful
- what a repository is
- what a commit is
- what a branch is
- what a remote is

## Required Tree

Include this tree in your own words:

```text
working directory
  -> staging area
    -> local repository
      -> remote repository on GitHub
```

## Required Example

Use a frontend example:

```text
I edit LoginPage.tsx.
I stage the change.
I commit the change with message "Add login form".
I push the commit to GitHub.
My teammate pulls the latest code.
```

## Demo Questions

Be ready to answer:

1. What is Git?
2. What is GitHub?
3. What is the difference between Git and GitHub?
4. What does a commit represent?
5. Why should teams avoid sending ZIP files?
