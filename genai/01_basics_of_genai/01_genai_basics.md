# Part 1: Generative AI Basics

This lesson explains the foundation of Generative AI for a beginner.

## 1. What Is Generative AI?

Generative AI is a type of artificial intelligence that creates new content.

That content can be:

- text
- images
- audio
- video
- code

Examples:

- writing an email draft
- summarizing a document
- creating an image from a text description
- generating code from a requirement
- turning text into speech

The model does not think like a human. It learns patterns from large amounts of
data and then generates output that matches those patterns.

## 2. Why Is It Called Generative?

It is called generative because it generates something new.

For example:

- a spam classifier tells you whether an email is spam or not spam
- a generative model can write a full email reply

So the main difference is:

- traditional predictive or classification AI chooses or labels
- generative AI creates

## 3. Generative AI vs Traditional AI

Traditional AI systems are often built to:

- classify
- predict
- detect
- recommend
- rank

Generative AI systems are built to:

- write
- answer
- summarize
- translate
- create images
- create voice or video
- generate code

Simple difference:

- traditional AI asks: "What is this?"
- generative AI asks: "What should I create next?"

## 4. Where Do We Use Generative AI?

Generative AI is used in many places:

- chat assistants
- coding assistants
- search and knowledge assistants
- customer support bots
- document summarization
- email drafting
- marketing content generation
- image design tools
- speech generation tools
- education tools
- healthcare documentation support

Many products already use GenAI in the background even if the user does not see
the model directly.

## 5. Common Types of Generative AI Output

### Text Generation

Examples:

- chatbot replies
- blog drafts
- summaries
- translations
- question answering

### Code Generation

Examples:

- function suggestions
- SQL query drafts
- code explanations
- test case generation

### Image Generation

Examples:

- posters
- product mockups
- concept art
- social media creatives

### Audio Generation

Examples:

- text-to-speech
- voice assistants
- voice cloning in approved use cases
- music generation

### Video Generation

Examples:

- short marketing clips
- educational animations
- synthetic training videos

## 6. What Makes GenAI Popular?

Generative AI became popular because it can help people work faster.

It can:

- reduce repetitive writing work
- speed up research
- help non-experts start quickly
- improve productivity
- support creativity

But popularity does not mean perfect reliability.

## 7. Simple Mental Model

You can think of GenAI like this:

1. The model has learned patterns from a lot of examples.
2. The user gives it a prompt.
3. The model predicts a useful next output.
4. The user reviews and improves that output.

This is why GenAI is often best used as a collaborator.

## 8. Important Beginner Truth

Generative AI can sound confident even when it is wrong.

That is why we must learn both:

- how to use it
- how to verify it

## 9. Short Summary

- Generative AI creates new content.
- It is used for text, code, images, audio, and video.
- It is different from systems that only classify or predict.
- It is useful in many industries and products.
- Human review is still important.
