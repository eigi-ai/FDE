# Part 3: Generative AI Use Cases

This lesson explains where Generative AI is used in real life and where it must
be used carefully.

## 1. Writing and Communication

Generative AI is widely used for text work.

Examples:

- email drafting
- blog writing support
- meeting summary generation
- report drafting
- rewriting content in a simpler tone
- translation and localization

This is useful because many jobs involve communication work.

## 2. Coding and Software Development

Developers use GenAI for:

- code suggestions
- debugging help
- test case generation
- documentation drafting
- SQL query generation
- API explanation

Important note:

AI-generated code must still be reviewed for:

- correctness
- security
- performance
- maintainability

## 3. Customer Support and Operations

Teams use GenAI for:

- chatbot responses
- ticket summarization
- agent assistance
- FAQ generation
- internal knowledge search

Many support systems combine GenAI with company documents using RAG.

## 4. Search and Knowledge Systems

GenAI is used to help users interact with information.

Examples:

- ask questions over PDF files
- search company documents in natural language
- summarize long policies
- compare multiple documents

This is one of the most useful business applications of GenAI.

## 5. Design and Media

GenAI is used for:

- image generation
- design variations
- ad creatives
- voiceovers
- music generation
- short video generation

This can speed up early ideation and creative iteration.

## 6. Education and Learning

GenAI can help learners by:

- explaining difficult concepts simply
- generating practice questions
- creating study summaries
- giving step-by-step examples
- converting long notes into quick revision points

But learners should still think independently and verify important information.

## 7. Healthcare, Finance, and Enterprise Work

In professional domains, GenAI can support:

- document drafting
- note summarization
- workflow assistance
- internal search
- form filling support
- data extraction from unstructured text

These areas require extra care because the cost of mistakes is high.

## 8. Where GenAI Should Be Used Carefully

Generative AI should be used carefully in cases such as:

- medical decisions
- legal advice
- financial decisions
- compliance workflows
- safety-critical systems
- sensitive personal data handling

In these cases, human review is necessary.

## 9. Benefits of GenAI

- saves time on repetitive work
- helps people start from a draft instead of a blank page
- improves productivity
- increases access to knowledge
- helps with creativity and experimentation

## 10. Limitations of GenAI

- may hallucinate facts
- may miss important context
- may reflect bias from training data
- may produce low-quality output from vague prompts
- may expose privacy or compliance risk if used carelessly
- may generate code or content that looks good but is wrong

## 11. Human Role in GenAI Systems

The best use of GenAI is usually human plus AI, not human replaced by AI.

Human responsibility includes:

- giving clear instructions
- checking the output
- correcting mistakes
- applying domain judgment
- deciding what is safe to share

## 12. Short Summary

- GenAI is used in writing, coding, support, search, design, education, and enterprise workflows.
- It creates real business value when paired with human review.
- It should be used carefully in high-risk domains.
- Good results depend on both technology and responsible usage.
