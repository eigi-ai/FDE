# Generative AI Tutorial For Beginners

This module teaches Generative AI from the beginning for learners who are new to
AI, machine learning, and modern AI-powered products.

Generative AI, often called GenAI, is a type of artificial intelligence that can
create new content such as text, images, audio, video, and code.

## How To Read This Module

Each topic is split into a separate chapter file.

Recommended way to learn:

1. Read the chapters in order.
2. Connect each concept to real products you already know.
3. Focus first on ideas, not model names.
4. Revisit prompts, RAG, and safety after understanding the basics.

## Important Terms

- AI: broad field of building systems that perform tasks that appear smart.
- Machine Learning: a way for systems to learn patterns from data.
- Generative AI: AI that creates new content.
- Model: the trained system that produces predictions or generated output.
- Prompt: the input instruction given to the model.
- Token: a small unit of text a language model reads and generates.
- LLM: Large Language Model, a model trained to work with text.
- Embedding: a numeric representation of meaning used for search and similarity.
- Inference: the stage where a trained model generates output.
- Fine-tuning: additional training to adapt a model to a specific domain or task.
- RAG: Retrieval-Augmented Generation, where external documents are retrieved and
  added to the prompt before generation.
- Agent: a system that uses a model plus tools and steps to complete tasks.
- Hallucination: when the model gives incorrect or made-up information.

## Beginner Notes

- Generative AI is useful, but it is not always correct.
- A good output depends on both the model and the prompt.
- AI answers should be reviewed before using them in production work.
- Do not put private company data into tools that are not approved for that data.
- GenAI usually works best as a helper, not as a fully trusted replacement for human judgment.
